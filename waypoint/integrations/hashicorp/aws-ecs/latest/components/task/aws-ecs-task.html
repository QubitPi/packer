AWS ECS

@hashicorp

The AWS ECS plugin deploys an application image to an AWS ECS cluster. It also launches on-demand runners to do operations remotely.

AWS ECS Task

Archive Notice

This integration relates to a legacy version of Waypoint and is no longer actively maintained.
Read the blog
(opens in new tab)

Launch an ECS task for on-demand tasks from the Waypoint server.

This will use the standard AWS environment variables and IAM Role information to source authentication information for AWS, using the configured task role. If no task role name is specified, Waypoint will create one with the required permissions.

Parameters

10 Results

  • clusterOptionalstringstring

    Cluster name to place On-Demand runner tasks in ECS Cluster to place On-Demand runners in. This defaults to the cluster used by the Waypoint server

  • execution_role_nameOptionalstringstring

    The name of the AWS IAM role to apply to the task's Execution Role ExecutionRoleName is the name of the AWS IAM role to apply to the task's Execution Role. At this time we reuse the same Role as the Waypoint server Execution Role.

  • log_groupOptionalstringstring

    Cloud Watch Log Group to use for On-Demand Runners Cloud Watch Log Group to use for On-Demand Runners. Defaults to the log group used for runners (waypoint-runner).

  • odr_cpuOptionalstringstring

    CPU to use for the On-Demand runners. Configure the CPU for the On-Demand runners. The default is 512. See https://docs.aws.amazon.com/AmazonECS/latest/developerguide/task_definition_parameters.html for valid values

  • odr_imageRequired

    Docker image for the Waypoint On-Demand Runners Docker image for the Waypoint On-Demand Runners. This will default to the server image with the name (not label) suffixed with '-odr'."

  • odr_memoryOptionalstringstring

    Memory to use for the On-Demand runners. Configure the memory for the On-Demand runners. The default is 1024. See https://docs.aws.amazon.com/AmazonECS/latest/developerguide/task_definition_parameters.html for valid values

  • regionOptionalstringstring

    AWS Region to use AWS region to use. Defaults to the region used for the Waypoint Server.

  • security_group_idRequiredstringstring

    Security Group ID to place the On-Demand Runner task in Security Group ID to place the On-Demand Runner task in. This defaults to the security group used for the Waypoint server

  • subnetsRequiredstringstring

    List of subnets to place the On-Demand Runner task in. List of subnets to place the On-Demand Runner task in. This defaults to the list of subnets configured for the Waypoint server and must be either identical or a subset of the subnets used by the Waypoint server

  • task_role_nameOptionalstringstring

    The name of the AWS IAM role to apply to the task's Task Role TaskRoleName is the name of the AWS IAM role to apply to the task. This role determines the privileges the ODR builder. If no role name is given, an IAM role will be created with the required policies

Outputs

  • idstringstring
